# Phase 62.2 Plan 01: Loops userGroup + OG Image + Session-Linked Lead Timeline Summary

**Loops contacts tagged with userGroup:"search", OG image replaced with 1200x627 landscape + summary_large_image card, and anonymous session searches linked to identified leads in admin timeline**

- **Started:** 2026-03-04

## Accomplishments
- All Loops contacts (email capture + newsletter) now include `userGroup: "search"` for audience segmentation
- Social media link previews use the correct 1200x627 landscape image with `summary_large_image` card type
- Newsletter subscribers have their `session_id` captured at signup, enabling pre-signup search activity to appear in the admin lead timeline
- Lead timeline endpoint queries `user_events` by linked `session_id` for `search_query` events, merging them into the chronological timeline

## Decisions Made
- userGroup "search" is an unconditional static field per user requirement
- First signup's session_id wins (on_conflict_do_nothing means re-subscribes are ignored)
- Used `query_text` payload key (matching frontend trackEvent format), not `query`
- Used `getattr(subscriber, 'session_id', None)` for safety during migration transition
