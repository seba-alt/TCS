## Accomplishments
- Leads table now shows click_count per lead in a dedicated "Clicks" column (purple badge for > 0, muted for 0)
- Clicks column header is clickable to toggle ascending/descending sort via useMemo-based local sort
- Flat Click Activity table below leads table shows all click events chronologically (email, expert, query, timestamp)
- Click Activity table limited to 50 most recent events with overflow indicator
- Empty state shows "No click activity yet" when no clicks exist
