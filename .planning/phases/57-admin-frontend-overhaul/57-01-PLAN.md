```
]},
```
3. Replace the flat `{ path: 'data', element: <DataPage /> }` with nested children:
```
{ path: 'data', element: <DataPage />, children: [

    { path: 'searches', element: <SearchesPage /> },
    { path: 'marketplace', element: <AdminMarketplacePage /> },
]},
```

**Lazy import pattern** — follow existing pattern in main.tsx:
