- Both sync functions (single + bulk rebuild) handle manual tags as ExpertTag rows with tag_type=manual
- FAISS embedding text merges manual_tags for semantic search benefit
- Public tag filter matches both skill and manual tag types
- Admin serializer deduplicates AI + manual tags and exposes manual_tags separately
