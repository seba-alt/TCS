   ```typescript
   export function useTagCatalog() {
     const [data, setData] = useState<TagCatalogResponse | null>(null)
     const [loading, setLoading] = useState(true)
     const [error, setError] = useState<string | null>(null)

     const fetchData = useCallback(() => {
       setLoading(true)
       adminFetch<TagCatalogResponse>('/tags/catalog')
         .then(setData)
         .catch((e: Error) => setError(e.message))
         .finally(() => setLoading(false))
     }, [])

     useEffect(() => { fetchData() }, [fetchData])

     return { data, loading, error, refetch: fetchData }
   }
   ```

**Top bar:** Mode toggle with two buttons/tabs (styled like ToolsPage tab navigation pattern):
- "Expert -> Tags" (default, active state = purple bg)
- "Tag -> Experts"

**Catalog management section** (shared between modes, displayed above the main content):
- Shows current predefined tag count
- "Add Tag" input + button (calls `POST /tags/catalog`, then refetches catalog)
- Tag list displayed as pills/chips with delete (x) button on each (calls `DELETE /tags/catalog/{id}`, then refetches)

**Mode A: Expert -> Tags:**
- **Left panel (w-1/2 or w-2/5):** Searchable expert list
  - Search input at top (client-side filter by name/username)
  - Each row: checkbox + name + job title + current manual tags as small chips
  - Select all / deselect all button
  - Use `useAdminExperts()` for expert data
- **Right panel (w-1/2 or w-3/5):** Tag assignment area
  - Header showing "N experts selected"
  - Predefined tag list as clickable chips/buttons (from catalog)
  - Click a tag to toggle selection (highlight selected tags)
  - "Assign Selected Tags" button (calls `POST /tags/assign` with selected usernames + tags, then refetches assignments and experts)
  - Below: "Current Tags" for selected expert (if single expert selected) — shows manual tags with remove (x) button (calls `DELETE /tags/assign/{username}/{tag}`)

**Mode B: Tag -> Experts:**
- **Left panel:** Tag list from catalog
  - Each tag shown as a clickable item
  - Click selects that tag as active
  - Show count of experts with each tag (from assignments data)
- **Right panel:** Experts assigned to selected tag
  - List of expert name + job title for all experts with this manual tag
  - "Add Experts" button that opens a searchable dropdown/list of unassigned experts
  - Remove button (x) per expert to un-assign (calls `DELETE /tags/assign/{username}/{tag}`)

**Key implementation details:**
- Use `useAdminExperts()` for the full expert list (includes active experts)
- Use `useTagCatalog()` for the predefined tag list
- Use `useTagAssignments()` for current manual tag assignments

- After any mutation, call `refetch()` on affected hooks
- All tag operations normalize to lowercase
